# cross-platform-yu-gothic

> @font-face to use **游ゴシック** in Cross-platform.

## Why?

 Cross-platform.

## Install

```
$ npm install cross-platform-yu-gothic --save
```

## Usage

```css
@import 'cross-platform-yu-gothic.css'

body {
	font-family: ;
}

```
